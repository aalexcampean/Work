Single Build Steps when present will be executed before the plug-in iterates the retrieved automated test cases. It is useful when you have a set of automated test cases and a single Build Step that executes a test suite, for instance.

List of environment variables that Jenkins TestLink Plug-in defines:

  • TESTLINK_BUILD_NAME
  • TESTLINK_TESTPLAN_NAME
  • TESTLINK_TESTPROJECT_NAME
  • TESTLINK_TESTCASE_TOTAL
